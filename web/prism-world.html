Prism World, a universe you can verify

One shared cellular universe. Its state is an integer grid with a content hash; its law is Prism source run through the WebAssembly interpreter, identified by the compiler's own Core hash. History is a branchable execution prefix. Open this page in a second tab: with no server between them, both tabs compute the same state hash for the same seed, law, and tick.

The Autoverse / Copies / Dust / Garden framing is an homage to Greg Egan's Permutation City. It is vocabulary, not a claim: every panel below is either backed by something that actually ran on your machine, or is labelled reserved / unchecked.
